# Library Management System - Frontend

JavaFX desktop application for the Library Management System.

## Prerequisites

- Java 25
- Maven 3.9+

## Dependencies

- JavaFX 23
- Jackson (JSON parsing)

## Running the Application

```bash
cd frontend
mvn javafx:run
```
